# Password_Generator_minor1
Made by: Gitanshu Sahoo, Roll ID: WD-FSWD-A4/May-10208

Project description: Password Generator is a web based software created by using HTML, CSS and Javascript. It produces passwords as per user's requirements how the password should be. It has 4 options: numbers, upper case, lower case and special case, based on user's choice of option(s), it generates a password of desired length and also tells about the strength
 of the password created. user can also copy the password by using copy button. This software has glassmorphism UI and a beautiful jellyfish background and floating bubbles animation and also has automatic dark/light mode as per device's setting for enhanced user experience.

 Tech used: HTML, CSS, Javascript, google photos (jellyfish background), google fonts (Poppins)

 Steps to use project: First, open the password generator software through the link, then enter the desired password length, then select what type(s) of characters you need in your password- lower case letters, upper case letters, numbers, special characters, then click on generate password, if you like the password, click on copy, if not, then click on generate password again.
